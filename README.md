Shopping Cart Website
A fully functional shopping cart website developed using PHP and MySQL, allowing users to sign up, log in, add items to a cart, manage the cart, and view the total amount.

Features
User Authentication: Secure login and registration system for user access.
Shopping Cart Management: Users can add items to their cart, remove items, and view total cart amount in real-time.
Product Listing: Displays available products with details for easy shopping.
Responsive Design: Works smoothly on desktops, tablets, and mobile devices.
